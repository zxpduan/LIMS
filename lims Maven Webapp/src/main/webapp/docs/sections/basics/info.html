帮助信息

帮助目录

1. 桌面

  • 打开系统后，缺省登录的界面。

2. 顶端导航信息

  • 顶端导航信息。可以在导航菜单设置

3. 顶端导航信息

  • 可将导航菜单收缩或打开

4.合同编号或者还款人模糊查找

  • 操作者输入合同编号，或者借款人姓名的首字母或者备注还款人姓名的首字母，系统会自动从数据库中搜索。缺省情况下，从数据库中随机取出20条数据备选。

4.合同编号或者还款人模糊查找

  • 所有标注"*"的必须输入。
  • 身份证号有合法性校验，非法的身份证是无法输入的。
  • 每项输入的数据长度是有限制的，特别是地址或者名称之类的，超出长度无法录入，系统会进行验证。
  • 年收入和购买价的单位是"万元"。
