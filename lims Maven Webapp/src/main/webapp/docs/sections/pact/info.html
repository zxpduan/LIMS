帮助信息

帮助目录

1. 合同录入

  • 打开系统后，缺省登录的界面。

2. 合同修改

  • 顶端导航信息。可以在导航菜单设置

3. 合同审核

  • 合同的状态有：审核中、审核通过、审核不通过和未审核4中状态。审核中、审核通过的合同无法修改的，只要超级管理员才有权处理。
  • 点击可以进行审核。
  • 审核通过够不能再次点击，通过审核的合同不能再次审核不通过，除非是超级管理员强制操作。

4.修改还款备注人

  • 操作者输入合同编号，或者借款人姓名的首字母或者备注还款人姓名的首字母，系统会自动从数据库中搜索。缺省情况下，从数据库中随机取出20条数据备选。

5.修改放款日

  • 所有标注"*"的必须输入。
  • 身份证号有合法性校验，非法的身份证是无法输入的。
  • 每项输入的数据长度是有限制的，特别是地址或者名称之类的，超出长度无法录入，系统会进行验证。
  • 年收入和购买价的单位是"万元"。
